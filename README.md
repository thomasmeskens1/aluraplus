#AluraPlus Clone / Projeto “Alura+”

Descrição

Este projeto é uma réplica funcional (UI/UX) do site da Alura+ (versão “Combo+”), disponível em https://aluraplus-gules-xi.vercel.app/

O objetivo é treinar e demonstrar habilidades em front-end (componentes, animações, etc.), reproduzindo a experiência de assinatura “Alura+ e Alura Língua”.

Funcionalidades principais

Layout semelhante ao do site original da Alura+: cabeçalho, hero section, principais vantagens (assista em vários dispositivos, baixe cursos etc.).
Responsividade adaptada para TV, desktop, tablet e celular.
Uso de imagens ilustrativas, ícones de dispositivo, grid ou flexbox para estrutura das sessões.
Deploy feito através de Vercel (endereço acima).
Design estático (sem back-end) — voltado para demonstração de UI.

Tecnologias utilizadas

HTML5 / CSS3 (ou alternativa: SCSS, CSS Modules, Styled Components)
Vercel para hospedagem e deploy contínuo
Controle de versão: Git + GitHub/GitLab
